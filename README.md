# B2B Product Catalog

Статическое веб-приложение для B2B каталога продукции с поддержкой PDF-экспорта и PWA.

## Быстрый старт

```bash
# 1. Сборка (конвертация CSV → JSON + генерация иконок)
python build.py

# 2. Запуск локального сервера
python -m http.server 8080 --directory dist
```

Откройте http://localhost:8080

## Обновление каталога

1. Отредактируйте `data/products.csv`
2. Запустите `python build.py`
3. Каталог обновлён

## Структура CSV

| Колонка          | Описание                    |
|------------------|-----------------------------|
| image            | Путь к изображению          |
| name             | Наименование товара         |
| description      | Описание                    |
| package          | Комплектация                |
| price_200        | Цена до 200 шт             |
| price_200_plus   | Цена от 200 шт             |
| price_500_plus   | Цена от 500 шт             |
| price_container  | Цена за контейнер           |

## Возможности

- Поиск по наименованию
- Адаптивный дизайн (mobile-first)
- Скачивание PDF-каталога (html2pdf.js)
- PWA: установка на телефон + офлайн-просмотр
- Только статика, без backend

## Деплой

Разместите содержимое папки `dist/` на любом статическом хостинге (Netlify, Vercel, GitHub Pages, nginx и т.д.).
